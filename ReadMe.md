## Описание проекта
Этот проект представляет собой экран приложения, реализованный с использованием Kotlin Compose. Экран представляет страницу Dota 2 и содержит следующие элементы:

- **Шапка**: Заголовок и логотип Dota 2.
- **Тэги**: Категории и характеристики игры.
- **Отзывы и оценки**: Пользовательские отзывы и оценки игры.
- **Скриншоты**: Скриншоты игры.
- **Кнопка "Установить"**: Для инициирования установки игры.

## Структура проекта
- **`MainActivity`**: Основной код экрана Dota 2.
- **`Description`**: Функция описания.
- **`DotaHeader`**: Функция шапки.
- **`DotaImage`**: Функция скриншотов Dota 2.
- **`DotaLogo`**: Функция логотипа.
- **`DotaName`**: Функция названия.
- **`DotaTag`**: Функция тэгов.
- **`InstallButton`**: Функция кнопки "установить".
- **`Rating`**: Функция рейтинга и "звездочек".
- **`Review`**: Функция отзывов.
- **`screenshots`**: Папка скриншотов приложения.

## Технологии
- Kotlin и Jetpack Compose: Для создания пользовательского интерфейса.
